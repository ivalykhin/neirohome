Что такое ИНС?

Искусственная нейронная сеть - это математическая модель, а также её программное или аппаратное воплощение, построенная по принципу организации и функционирования биологических нейронных сетей — сетей нервных клеток живого организма.

  • Описание ИНС на Wikipedia

Исследование искусственных нейронных сетей для предсказаний котировок в будущем

Прогнозирование является одной из задач, которые может выполнять ИНС. В случае, который рассматривается на данном сайте точность прогнозирования будет одной из основных метрик при оценке качества ИНС. Данный сайт является интерфейсом для отображения информации об успехах и неудачах ИНС в этом нелегком деле.

Виды искусственных нейронных сетей

В данный момент все ИНС являются линейными многослойными сетями с 1 выходом, который выводит результат прогнозирования.

Способы обучения

Обучение сетей производится с помощью учителя. В данный момент используются способы обучения: Обратного распространения ошибки (Backprop), Resilient Propagation (RProp)
